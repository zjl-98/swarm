# swarm
flask 爬虫项目集（虫聚）

# 版本
v1.0

# 功能
1. flask前端界面显示
2. 数据库存储设计
3. 影视热数据、壁纸热数据、影视搜索、壁纸下载功能实现
4. 用户登录注册、邮箱验证功能实现

# 文件
app --项目工程文件
  forms --校验器
  libs --工具文件
  models --数据库模型
  spider --爬虫文件
  static --静态文件
  templates --模板文件
  view_models --视图文件
  web --flask蓝图
  setting.py --外显配置文件
swarm.py --项目入口文件
